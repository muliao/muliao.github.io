把搭建这个blog环境的攻略作为第一篇博客

October 9, 2018 - 1 minutes read - 85 words

记录下这个简单的blog环境的搭建过程。

使用hugo + ananke主题

对它的预期是希望做到几点就好。1：支持makedown格式编辑；2：支持代码高亮；3：支持大纲；4：支持评论；5：简单的发布

hugo搭建

之前按照jane搭建的,一年后重新用时发现会报错，决定还是用教程推荐的ananke主题

参考hugo的quick start进行安装和新建站点

参考sample里面的config.tom,做些简单的配置就可以支持自己写点简单的记录了。

修改config.tom中的个人定制信息

做一些必要功能的修改

languageCode = "zh-cn"  #默认中文
DefaultContentLanguage = "cn" #默认中文
Paginate = 10 #一次多显示一些dummy contetn

[params]
  recent_posts_number = 5 #多显示一些
  commentoEnable = true #打开评论
  show_reading_time = true  #显示字数
  background_color_class = "bg-blue"  #显示背景颜色

预览测试

执行下面指令，然后访问 http://localhost:1313 来预览

hugo -D server

发布博客到自己的github上

在最前面已经创建好了仓库了,现在只要克隆到muliao_blug/public目录下

$ cd muliao_blog
$ git clone https://github.com/muliao/muliao.github.io.git public
$ hugo
$ cd public
$ git add .
$ git commit -m '发表了第一篇文章'
$ git push

hugo指令生成静态页面到public，此时上传这些文件即可

访问 https://muliao.github.io 是不是可以看到自己的文章了呢。
