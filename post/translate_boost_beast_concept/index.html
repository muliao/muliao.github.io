DynamicBuffer 动态缓冲区

A dynamic buffer encapsulates memory storage that may be automatically resized as required, where the memory is divided into an input sequence followed by an output sequence.
动态缓冲区封装了可以按要求自动调整大小的内存存储，其中内存分为输入序列和输出序列。
These memory regions are internal to the dynamic buffer, but direct access to the elements is provided to permit them to be efficiently used with I/O operations, such as the send or receive operations of a socket.
这些内存区域在动态缓冲区内部，但是提供了对元素的直接访问，以允许它们与I / O操作（例如套接字的发送或接收操作）有效地结合使用。
Data written to the output sequence of a dynamic buffer object is appended to the input sequence of the same object.
写入动态缓冲区对象的输出序列的数据将附加到同一对象的输入序列。

The interface to this concept is intended to permit the following implementation strategies:
此概念的接口旨在允许以下实施策略：

  • A single contiguous octet array, which is reallocated as necessary to accommodate changes in the size of the octet sequence. This is the implementation approach currently offered by flat_buffer.
    单个连续的字节数组，可根据需要重新分配以适应八位位组序列大小的变化。这个实现由flat_buffer提供

  • A sequence of one or more octet arrays, where each array is of the same size. Additional octet array objects are appended to the sequence to accommodate changes in the size of the octet sequence.
    一个或多个字节数组的序列，其中每个数组的大小相同。其他字节数组对象将附加到序列中，以适应字节数组序列大小的变化。

  • A sequence of one or more octet arrays of varying sizes. Additional octet array objects are appended to the sequence to accommodate changes in the size of the character sequence. This is the implementation approach currently offered by multi_buffer.
    一个或多个大小不同的八位字节数组的序列。其他八位位组数组对象将附加到序列中，以适应字符序列大小的变化。这是所提供的实施方法multi_buffer。

Requirements 要求

  • D denotes a dynamic buffer class.
    D表示动态缓冲区
  • a denotes a value of type D.
    a表示D类型的值
  • c denotes a (possibly const) value of type D.
    c表示D类型的(可能是const)值
  • n denotes a value of type std::size_t.
    n表示std::size_tl类型的值
  • T denotes a type meeting the requirements for ConstBufferSequence.
    T表示满足ConstBufferSequence要求的类型。
  • U denotes a type meeting the requirements for MutableBufferSequence.
    U表示满足MutableBufferSequence要求的类型。

Table 1.31. Valid expressions

Expression Type Semantics, Pre/Post-conditions
D::const_buffers_type T This type represents the memory associated with the input sequence.
此类型表示与输入序列关联的内存。
D::mutable_buffers_type U This type represents the memory associated with the output sequence.
此类型表示与输入序列关联的内存。
c.size() std::size_t Returns the size, in bytes, of the input sequence.
返回输入序列的大小（以字节为单位）。
c.max_size() std::size_t Returns the permitted maximum of the sum of the sizes of the input sequence and output sequence.
返回输入序列和输出序列的大小之和的允许最大值。
c.capacity() std::size_t Returns the maximum sum of the sizes of the input sequence and output sequence that the dynamic buffer can hold without requiring reallocation.
返回动态缓冲区无需重新分配即可容纳的输入序列和输出序列的大小的最大和。
c.data() D::const_buffers_type Returns a constant buffer sequence u that represents the memory associated with the input sequence, and where buffer_size(u) == size().
返回一个常量缓冲区序列u，它表示与输入序列关联的内存，其中。 buffer_size(u) == size()
a.prepare(n) D::mutable_buffers_type Returns a mutable buffer sequence u representing the output sequence, and where buffer_size(u) == n. The dynamic buffer reallocates memory as required. All constant or mutable buffer sequences previously obtained using data() or prepare() are invalidated.
返回表示输出序列的可变缓冲区序列u，其中 buffer_size(u) == n。动态缓冲区根据需要重新分配内存。先前从data()或prepare()获取的所有恒定或可变缓冲区序列均无效。
Throws: length_error if size() + n exceeds max_size().
如果size()+n超过max_size(),抛出length_error
a.commit(n) Appends n bytes from the start of the output sequence to the end of the input sequence.
把输出队列起始的n个字节追加到输入队列尾部The remainder of the output sequence is discarded.
输出序列的其余部分将被丢弃。
If n is greater than the size of the output sequence, the entire output sequence is appended to the input sequence.
如果n大于输出序列的大小，则将整个输出序列附加到输入序列中。
All constant or mutable buffer sequences previously obtained using data() or prepare() are invalidated.
先前从data()或prepare()获取的所有恒定或可变缓冲区序列均无效。
a.consume(n) Removes n bytes from beginning of the input sequence.
从输入队列开头删除n个字节
If n is greater than the size of the input sequence, the entire input sequence is removed.
如果n大于输入队列的大小，则整个队列被删除
All constant or mutable buffer sequences previously obtained using data() or prepare() are invalidated.
先前从data()或prepare()获取的所有恒定或可变缓冲区序列均无效。
